## Writing Rules for COWORK Claude

1. **수치 정확성**: 모든 숫자는 `statistics.md`에서 직접 인용. 절대 추측하지 말 것
2. **Figure/Table 참조**: "Fig. 1", "Table I" (IEEE) 또는 "Figure 1", "Table 1" (기타 저널)
3. **단위**: SI 단위 사용 (N, m/s, deg, %, s)
4. **통계 표기**: "mean ± SD", "p < 0.05", "Cohen's d = 0.45"
5. **참고문헌**: `references.bib`의 BibTeX key 사용 (e.g., \cite{winter1992biomechanics})
6. **톤**: 학술적, 객관적, 3인칭
7. **약어**: 첫 사용 시 풀네임 (e.g., "gait cycle percentage (GCP)")
