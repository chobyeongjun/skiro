## 계획 대비 분석

### 목표별 달성 평가
계획 단계에서 세운 각 목표에 대해:
```
목표: <원래 목표>
결과: <실제 결과>
달성: YES / PARTIAL / NO
원인: <달성/미달성 원인>
```

### 시간 추정 정확도
```
계획: N분 / 실제: M분 / 오차: ±X%
반복 패턴: 항상 과소추정 or 과대추정 → 다음 계획 시 보정 계수 적용
```

### 롤백 실행 여부
```
롤백 실행: YES / NO
실행 이유: <이유>
롤백 절차 적절성: 계획대로였나?
```
