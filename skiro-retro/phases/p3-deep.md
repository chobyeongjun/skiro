## 실험 데이터 회고

### 결과 요약 통계
```
N회 시행 중:
  - 성공: N회 (X%)
  - 실패: M회 (Y%)
  - 이상치: K회 (제외 기준: <기준>)

주요 측정값:
  - 평균 ± 표준편차
  - 중앙값 (IQR)
  - 최대/최소
```

### 기대값 대비 평가
```
기대:   <계획에서 설정한 기준>
실제:   <측정값>
판정:   통과 / 미달 / 초과
원인:   <분석>
```

### 데이터 품질 평가
```
□ 이상치 처리 방법 기록
□ 센서 오류 구간 제거 여부
□ 동기화 문제 (타임스탬프 불일치)
□ 환경 변수 통제 충분성
```
